# SmartBreed Lab

面向育种数据专家/数字育种研发岗位制作的静态概念验证网站。

## 功能

- 核心群遗传评估看板
- PBLUP、GBLUP、ssGBLUP 模型切换与模拟结果
- 多性状综合选择指数设计
- 近交约束下的智能选配建议
- 数字育种平台分层架构与 90 天落地路线
- 候选人岗位能力证据

## 运行

直接打开 `index.html`，或在当前目录启动任意静态服务器：

```powershell
python -m http.server 8000
```

访问 `http://localhost:8000`。

## 说明

本项目为求职作品集原型，不是福建永诚育种的官方系统。全部业务数据为模拟/脱敏数据，不用于生产决策。
